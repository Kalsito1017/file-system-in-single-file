# Файлова система върху единичен файл

Проектът представлява симулация на файлова система, реализирана върху един-единствен контейнерен файл. 
Поддържа основни файлови операции като създаване, изтриване, копиране, директории, дефрагментация и навигация.

Проектът е реализиран на **C# (.NET)** и работи чрез конзолен интерфейс.

---

##  Основна идея

Всички файлове и директории се съхраняват вътре в един файл (`.fsc`), който съдържа:

- хедър с информация за версията
- таблица с метаданни
- списък със свободни блокове
- реалните данни на файловете
- поддръжка на директории
- компресия и проверка за целостта

---

##  Поддържани команди

| Команда | Описание |
|------|----------|
| `cpin <външен_файл> <име>` | Копира файл в контейнера |
| `cpout <име> <външен_файл>` | Извлича файл от контейнера |
| `ls` | Показва съдържанието на текущата директория |
| `md <име>` | Създава директория |
| `rd <име>` | Изтрива директория |
| `cd <път>` | Смяна на директория |
| `rm <име>` | Изтрива файл |
| `defrag` | Дефрагментира контейнера |
| `info` | Информация за контейнера |
| `help` | Показва помощ |
| `exit` | Изход от програмата |

---

##  Основни компоненти

### `FileSystemContainer`
Отговаря за:
- управление на метаданните
- навигация между директории
- четене и запис на файлове
- изтриване
- дефрагментация

### `BlockManager`
- управлява свободните блокове
- разпределя и освобождава място
- поддържа списък със свободни блокове

### `MetadataEntry`
- описва файл или директория
- съдържа размер, име, родител, времеви маркери и др.

---

##  Основни концепции

- контейнерна файлова система  
- работа с бинарни файлове  
- управление на свободна памет  
- дефрагментация  
- компресия  
- контрол на целостта (checksum)  
- навигация тип UNIX  
- конзолен интерфейс
